Refonte PHP : Tableaux en Colonnes pour Performance et Responsive


L'entreprise avais besoin un changement significatif dans le code PHP qui générait les pages professionnelles de plusieurs de ses sites web. Avant cette modification, l'affichage de ces pages sur les téléphones portables était médiocre, ce qui affectait négativement l'expérience utilisateur. En effet, le contenu était structuré sous forme de tableaux, une mise en page peu adaptée aux petits écrans des smartphones.

Pour remédier à ce problème, l'équipe a repensé entièrement l'affichage en adoptant une structure en colonnes. Cette nouvelle approche a permis une meilleure adaptabilité et une lecture plus fluide sur les appareils mobiles. En organisant le contenu en colonnes plutôt qu'en tableaux, les informations étaient présentées de manière plus claire et plus accessible, offrant ainsi une expérience utilisateur optimisée.

Période : Stage 1er année.


  • Fréquence des sauvegardes : 1 fois pars semaine
  • Sauvegarde automatique en cronjob (akeeba pro)
  • Type de sauvegarde : complète(ftp+bdd+setups (.ht, versions php/bdd, etc.))
  • Emplacement : VPS OVH (debian)
