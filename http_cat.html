HTTP CAT

Présentation du Projet


J’ai créé un jeu éducatif pour Android en utilisant Android Studio. Dans ce jeu, un texte explicatif sur un code HTTP particulier s'affiche, et l'utilisateur doit entrer le numéro de code correspondant.


L'application a pour but de fournir une expérience éducative interactive et peut fonctionner sans connexion internet. Cela permet aux utilisateurs d'apprendre et de réviser les codes HTTP à tout moment et en tout lieu.

Période : 2eme année.


Information complémentaire

  • Différentes versions stables sauvegardées précédemment.
